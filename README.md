
🏦 Python Bank Account System

A simple command-line bank account system built using Python.
This project demonstrates Object-Oriented Programming (OOP), decorators, and exception handling.

🚀 Features

- Check account balance
- Deposit money
- Withdraw money
- Transaction logging using decorators
- Input validation with exception handling
- Interactive menu system

🛠 Technologies Used

- Python 3
- Object-Oriented Programming
- Decorators
- Exception Handling

📂 Project Structure

bank_account.py → Main program file

▶️ How to Run

1. Download or clone the repository
2. Open terminal in the project folder
3. Run the program:

python bank_account.py

📚 What I Learned

- Creating classes and methods
- Using Python decorators
- Handling exceptions
- Building a simple command-line application
